# Haltern App – Presentation

This repository contains a presentation for a potential **city app for Haltern am See**.

The slides present a concept for a locally developed mobile app and compare it with typical off-the-shelf city app platforms.

## Topics

- Idea and motivation for a city app
- Custom solution vs. off-the-shelf platforms
- Possible features for residents and visitors
- Visual concept of a mobile app
- Discussion points for future development

## Example Features

- City information and office hours
- Push notifications
- Issue reporting (Mängelmelder)
- Event calendar
- Waste collection calendar
- Citizen participation
- Local business and tourism information

## Notes

This repository only contains the **presentation material** and does not represent a finished software project.
